# Images Folder

## Required Images

Place your portfolio project images in this folder with the following names:

1. **passport-desktop.jpg** - Passport Photo Toronto desktop screenshot
2. **passport-mobile.jpg** - Passport Photo Toronto mobile screenshot
3. **headshot-desktop.jpg** - Headshot Photo Toronto desktop screenshot
4. **headshot-mobile.jpg** - Headshot Photo Toronto mobile screenshot
5. **julian-photo-desktop.jpg** - Julian Ross Photography desktop screenshot
6. **julian-photo-mobile.jpg** - Julian Ross Photography mobile screenshot
7. **julian-music-desktop.jpg** - Julian Ross Music desktop screenshot
8. **julian-music-mobile.jpg** - Julian Ross Music mobile screenshot

## Image Specifications

- **Format:** JPG, PNG, or WebP
- **Desktop images:** Recommended width 1200-1600px
- **Mobile images:** Recommended width 400-600px
- **Optimization:** Compress images for web to reduce file size

## Temporary Placeholder

If you don't have images yet, you can use placeholder services:

```html
<!-- Example placeholder -->
<img src="https://via.placeholder.com/1200x750/DC143C/FFFFFF?text=Your+Project" alt="Project Name">
```

Or create simple colored rectangles as placeholders until you have real screenshots.

## Adding New Images

1. Add image file to this folder
2. Reference it in index.html: `<img src="images/your-image.jpg" alt="Description">`
3. Make sure the filename matches exactly (case-sensitive)
